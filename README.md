# ptd
ptd is an interface for connections between the Pterodactyl daemon and Discord.

### Configuration
```js
 {
	"discord": {
		"token": "",
		"prefix": "",
		"privRole": "123456789012345678"
	},
	"panel": {
		"endpoint": "", 
		"public": "",
		"private": ""
	},
	"server": {
		"uuid": "",
		"ip": "",
		"port": 25565
	}
}
```
